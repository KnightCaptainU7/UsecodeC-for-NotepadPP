Usecode C user language and autocomplete.

Due to limitations of Notepad++, the AutoComplete file name must match the language name in the user language file.
usecodec.xml should go in to \Program Files\Notepad++\autoCompletion\

Then in Notepad++, import the user language file usecodec_userlanguage.xml
You may need to close and reopen Notepad++ for the custom language list entry to appear.
